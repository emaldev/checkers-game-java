# ♟️ Checkers Game - Java

A console-based Checkers game built with Java.

This project is a simple implementation of the classic Checkers board game using Java programming concepts and Object-Oriented Programming (OOP).

The goal of this project was to practice game logic, class design, and problem-solving skills.

## 🎮 Features

- Two-player console gameplay
- Board representation
- Player turn management
- Piece movement logic
- Capture moves
- Win condition checking
- Object-Oriented design

## 🛠 Technologies

- Java
- Object-Oriented Programming (OOP)
- Data Structures
- Git & GitHub

## 📚 Concepts Practiced

- Classes and Objects
- Encapsulation
- Methods and Constructors
- Arrays
- Game Logic Implementation
- Problem Solving
